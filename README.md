# claims_generator_backend
Grelin Core Claims Generator is a backend engine that automates creation, validation, formatting, and submission of insurance claims. It uses a rule-based system to generate compliant EDI 837, JSON, and XML outputs, integrates with external payer APIs, handles drafts and submissions, and ensures HIPAA-compliant data management.

# Key Features

Automated claim status checks

Integration with clearinghouses/payers

Stores verification results with timestamps

Robust error handling & logging

Easily extensible for additional payers

# Tech Stack

Node.js – Backend runtime

Express.js – API framework

Sequelize – ORM for database operations

PostgreSQL – Primary data store

Axios – External API communication

Winston – Structured logging

# Purpose

Streamlines claim tracking, reduces manual follow-up, and provides accurate, real-time visibility into claim status for healthcare workflows.
